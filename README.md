## ⬇️ **Installation**

1. Clone this repo

   HTTPS

   ```sh
   git clone https://github.com/dolbilov/BasicDataTemplateSample.git
   ```

   SSH

   ```sh
   git clone git@github.com:dolbilov/BasicDataTemplateSample.git
   ```

   GitHub CLI

   ```sh
   gh repo clone dolbilov/BasicDataTemplateSample
   ```

2. Restore dependencies

   ```sh
   dotnet restore
   ```

3. Run project
   ```sh
   dotnet run
   ```
